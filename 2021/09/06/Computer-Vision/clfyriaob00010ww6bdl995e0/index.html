FlowNet2.0-pytorch安装过程

发表于 2021-09-06 分类于 Computer Vision

官方提供的FlowNet2.0-pytorch的安装指南。

代码地址：https://github.com/NVIDIA/flownet2-pytorch
环境配置：（官方代码中所需环境为python36+pytorch0.4.1，但亲测0.4.1以上~1.1.0以下版本也可行）
 - python 3.7
 - pytorch 1.0.1
 - cudatoolkit 11.0
系统：Ubuntu 20.04.1

  进入/usr/local文件夹下，有时可能不存在cuda文件夹。此时打开终端，输入nvcc -V，若可以显示nvcc版本，则可能的原因参考https://blog.csdn.net/ai_faker/article/details/111687801。
